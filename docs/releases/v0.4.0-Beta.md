# sxPanel v0.4.0-Beta

This is the first sxPanel-branded beta release.

## Highlights

- Full project rebrand from fxPanel to sxPanel across the web panel, in-game menu, Discord bot, addon SDK, docs, and release metadata.
- New 0.4.0 beta panel work: queue tooling, whitelist workflows, deferral card builder, addon starter template, diagnostics updates, and player/admin management improvements.
- Release artifact is the generated `monitor.zip`; replace the FXServer `monitor/` resource with its contents.

## Notes

- This is a prerelease build.
- Existing txAdmin/sxPanel `txData` profiles should remain compatible.
- The release zip includes `THIRD-PARTY-LICENSES.txt` generated during the build.
